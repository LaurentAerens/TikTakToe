# Security Policy

## Supported Versions

Only the latest release line is actively supported with security fixes.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| older   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities **privately** using [GitHub's private security advisory feature](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability) in this repository.

When submitting, include:

- A clear description of the vulnerability.
- Steps to reproduce.
- Impact and affected version(s).
- Any suggested remediation, if available.
